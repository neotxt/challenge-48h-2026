# Challenge 48H — Messagerie & Gestion

## 👤 Mayles — Messagerie + Gestion

### Stratégie
- **Gestion de projet**
- **Gestion du budget**

### Fonctionnalités développées

- **Système de messagerie**  
    Permet aux utilisateurs d’échanger des messages en temps réel ou différé.

- **Base de données**  
    - **MCD (Modèle Conceptuel de Données)**
    - **Tables SQL** pour la gestion des utilisateurs, messages.

- **Lien vers le job board**  
    Intégration d’un accès direct au job board pour faciliter la navigation.

---

## Structure du projet

- `/messagerie` : Code source du système de messagerie
- `/database` : Scripts SQL et MCD
- `/job-board-link` : Lien ou composant d’intégration

---

## Installation

1. Cloner le dépôt
2. Installer les dépendances nécessaires
3. Configurer la base de données avec les scripts fournis
4. Lancer l’application
